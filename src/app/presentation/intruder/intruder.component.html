Intruder
About

Intruder is a cloud-based vulnerability scanner and security assessment tool designed for organizations of all sizes. It helps businesses identify and prioritize vulnerabilities in their IT infrastructure, web applications, and APIs, and provides actionable insights to help teams remediate security issues quickly.

The platform also includes features such as asset discovery, which automatically identifies all assets on your network, as well as integrations with popular security tools like Slack and JIRA to help streamline the vulnerability management process.
